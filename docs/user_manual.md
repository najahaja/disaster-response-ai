# Disaster Response AI - User Manual

## 📖 Introduction

Welcome to the Disaster Response AI system! This platform allows you to simulate and analyze AI-powered disaster response operations using multi-agent reinforcement learning.

### 🎯 What You Can Do

1. **Simulate disasters** in realistic urban environments
2. **Coordinate teams** of drones, ambulances, and rescue teams
3. **Monitor performance** with real-time analytics
4. **Train AI agents** using reinforcement learning
5. **Test strategies** across different scenarios

## 🚀 Quick Start Guide

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/your-username/disaster-response-ai.git
cd disaster-response-ai

# 2. Create virtual environment (recommended)
python -m venv disaster_env
source disaster_env/bin/activate  # Windows: disaster_env\Scripts\activate

# 3. Install dependencies
pip install -r requirements.txt

# 4. Verify installation
python test_complete_system.py
```
